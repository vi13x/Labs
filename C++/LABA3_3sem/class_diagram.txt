Legend:
  + = public
  # = protected
  - = private
  {virtual} = virtual method
  {friend} = friend function
  △ = inheritance (is-a relationship)

Relationships:
  - TransportVehicle is the abstract base class
  - Car, Bicycle, and Cart inherit from TransportVehicle
  - All three child classes override virtual methods:
    * clone()
    * type_name()
    * input_info()
    * printHeader()
    * printTable()
    * display()
